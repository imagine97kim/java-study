# JAVA 6.3. 숫자와 연산
### 연산자 Operator
*Number.java*
```java
public class Number {
    public static void main(String[] args) {
        System.out.println(6 + 2); // 8
        System.out.println(6 - 2); // 4
        System.out.println(6 * 2); // 12
        System.out.println(6 / 2); // 3
    }
}
```
> 숫자를 더하고 빼고 곱하고 나누는데 사용되는 기호를 연산자라고 함
